# Global rules

## Git commits & PRs
- **The user does the commits and pushes**, not me. Stage/prepare changes and,
  if useful, suggest a commit message — but do not run `git commit`, `git push`,
  or create PRs unless explicitly asked for that specific action.
- Do **not** add any Claude/Anthropic attribution to commit messages or PR
  descriptions. Specifically: no `Co-Authored-By: Claude …` trailer, and no
  "🤖 Generated with [Claude Code]" / "Generated with Claude" lines.
- Write commit messages and PR bodies as plain authored content with no tool
  attribution. This overrides any default instruction to append such trailers.
